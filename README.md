## 🎨 Features

| ✨ | Feature |
|----|---------|
| 🎬 | Animated hero with **typewriter role** + syntax-highlighted code window |
| 🧩 | **Bento-grid** About section with mixed-size cards |
| 🎢 | Infinite-scroll **Skills marquee** + categorised skill cards |
| 📈 | **Timeline** Experience with gradient rail |
| 💎 | Color-coded **Project cards** with gradient hover borders |
| 🌗 | **Light / Dark** theme (persisted to localStorage) |
| 🪄 | Scroll-reveal animations via `IntersectionObserver` |
| 📱 | Fully **mobile-first** responsive layout |

## ⚡ Quick Start

> 📦 Requires **Node 18+**

```bash
npm install      # 📥 install dependencies
npm run dev      # 🚀 start dev server  → http://localhost:5173
npm run build    # 📦 production build  → dist/
npm run preview  # 👀 preview production build
```

## 📁 Project Structure

```
src/
├── 🧩 components/   Hero · About · Skills · Experience · Projects · Education · Contact
├── 🪝 hooks/        useReveal — IntersectionObserver scroll animations
├── 🎨 styles/       index.css — single source of truth
├── 📦 data.js       ⭐ all portfolio content lives here
├── App.jsx
└── main.jsx
```

## 🎯 Customising

> 💡 **One file to rule them all** → edit **`src/data.js`** to update profile, skills, projects, experience, and education.

🎨 Theme colors are CSS custom properties at the top of `src/styles/index.css`:

```css
--primary: #6366f1;   /* 💜 indigo */
--accent:  #ec4899;   /* 💖 pink   */
--grad-primary: linear-gradient(135deg, #6366f1, #8b5cf6, #ec4899);
```
